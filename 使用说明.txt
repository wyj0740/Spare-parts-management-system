备品备件管理系统 - 使用说明
==========================

1. 软件启动
-----------
- 双击运行 "备品备件管理系统.exe"。
- 首次运行会自动创建数据库文件 (data 目录) 和必要目录。
- 程序启动后会自动打开浏览器并跳转到系统主页。
- 程序会最小化到系统托盘（右下角），右键托盘图标可退出。
- 同一时间只能运行一个程序实例，重复运行会自动打开浏览器。

2. 登录信息
-----------
- 默认账号：admin
- 默认密码：admin
- 可以在 config.ini 中修改默认账号密码。

3. 文件管理
-----------
- 每个备件会自动创建独立的文件夹。
- 点击"打开文件夹"按钮可在Windows资源管理器中管理文件。
- 文件存放在 files 目录下：
    - files/spare_parts/     备件相关文件
    - files/historical_documents/  历史文档

4. 部署与迁移
-------------
- 如需移动软件到其他文件夹或电脑，请【务必】复制整个文件夹。
- 必须包含：
    - 备品备件管理系统.exe
    - _internal 文件夹 (包含所有核心运行库)
    - templates 文件夹 (界面模板)
    - static 文件夹 (样式和脚本)
    - config.ini (配置文件)
- 数据文件：
    - data 文件夹 (数据库)
    - files 文件夹 (所有文件)
- 建议将整个文件夹压缩后发送。

5. 数据备份
-----------
- 系统内置自动备份功能，默认每天凌晨 2:00 备份。
- 也可以在系统的"数据备份"页面手动执行备份。
- 数据库文件存放在 data 目录下，请定期备份。
- 备份文件存放在 backups 目录下。

6. 注意事项
-----------
- 请确保路径中不要包含过长的特殊中文字符或符号。
- 建议放在 D 盘等非系统盘，避免权限限制。
- 如果无法启动，请检查 logs 目录下的日志文件。
- 端口默认5000，如被占用请修改 config.ini。
